# 🐧 Debian Installation (proot)

---

## 🔹 Install Debian

```bash
proot-distro install debian
```

---

## 🔹 Enter Debian

```bash
proot-distro login debian
```

---

## 🔹 Verify Installation

```bash
cat /etc/os-release
```

Expected:

```
Debian GNU/Linux
```

---

## 🔹 Update System

```bash
apt update && apt upgrade -y
```

---
