# 🧪 Tests

This directory contains a basic script to verify that the System Health API is working correctly.

---

## ▶️ Test Script

`test.sh`

A simple shell script that sends a request to the API endpoint and prints the response.

---

## ⚙️ Setup

Make the script executable:

```bash
chmod +x tests/test.sh
```

---

## 🚀 Run Test

```bash
./tests/test.sh
```

---

## 📌 Expected Output

- API response in JSON format
- Confirmation message indicating successful test

---

## 🧠 Purpose

- Quick verification of API availability
- Helps ensure the system is running correctly
- Useful during development and debugging
